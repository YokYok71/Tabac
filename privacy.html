Politique de confidentialité

Ma Cave à Tabac — t-cellar.app  ·  Dernière mise à jour : avril 2026

Résumé

Ma Cave à Tabac est une application web progressive (PWA) personnelle. Elle ne collecte aucune donnée utilisateur sur un serveur. Toutes vos données restent sur votre appareil.

Données stockées localement

L'application stocke les informations suivantes exclusivement sur votre appareil via localStorage :

  • Votre collection de tabacs (noms, marques, notes, lots, photos)
  • Votre collection de pipes
  • Votre liste de souhaits
  • Vos préférences d'interface (langue, clé API)

Ces données ne quittent jamais votre appareil sauf si vous utilisez la sauvegarde Google Drive (voir ci-dessous).

Google Drive (sauvegarde optionnelle)

Si vous choisissez d'utiliser la fonctionnalité de sauvegarde, l'application demande l'accès au scope drive.appdata de votre compte Google. Ce scope permet uniquement :

  • De lire et écrire un fichier de sauvegarde (cave-tabac-backup.json) dans un dossier privé réservé à l'application, invisible dans votre Google Drive

L'application n'accède à aucun autre fichier de votre Google Drive. L'authentification est gérée directement par Google via OAuth 2.0. Aucun token ni donnée Google n'est transmis à un tiers.

Assistant IA (optionnel)

Si vous configurez une clé API (Anthropic, OpenAI ou Google Gemini), l'application envoie directement depuis votre navigateur des requêtes aux API correspondantes pour l'auto-complétion des fiches tabac et pipe. Seules les informations saisies dans le champ de recherche sont transmises. La clé API est stockée localement sur votre appareil et n'est jamais envoyée à un autre service.

Aucun serveur tiers, aucun tracking

Ma Cave à Tabac est hébergée sur GitHub Pages. L'application ne dispose d'aucun serveur propre, ne pose aucun cookie de tracking et n'intègre aucun outil d'analyse (Google Analytics, etc.).

Plateforme

L'application fonctionne sur navigateur web (Safari, Chrome, Firefox, etc.) et peut être installée comme PWA sur l'écran d'accueil iOS et Android. Il n'existe pas de version native iOS ou Android distribuée via l'App Store ou le Play Store.

Contact
