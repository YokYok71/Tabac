1. Concepts de base

L'application s'organise autour de deux objets principaux :

  • Tabac — une référence (ex. : Dunhill Early Morning Pipe). Un tabac peut avoir zéro ou plusieurs lots.
  • Lot — une boîte ou quantité précise achetée à une date donnée. C'est le lot qui porte le statut et l'historique de vieillissement.

Cette distinction permet de suivre plusieurs boîtes d'un même tabac à des stades différents (une en cave, une en pot) tout en conservant un seul espace de notes et de description.

2. Cycle de vie d'un tabac

→

→
🫙 Pot Boîte ouverte
→
✅ Terminé Lot fini

📦 Cave (Cellar)

Un lot nouvellement acheté entre directement en Cave. Il représente une boîte fermée qui vieillit. L'âge affiché est calculé depuis la date de production (si renseignée) ou depuis la date d'achat.

Champs disponibles à ce stade :

ChampDescription
Poids (g)Quantité en grammes (ex. : 50, 100, 454)
Date d'achatPré-remplie avec la date du jour
Date de productionPour le calcul d'âge précis — reportez-la depuis la boîte
N° boîteNuméro d'identification si vous en avez plusieurs du même tabac
Prix (CHF)Prix d'achat
VendeurBoutique ou revendeur

🫙 Pot (Jar)

Quand vous ouvrez une boîte pour la mettre en pot, passez le lot en Pot. Appuyez sur le bouton ✏️ du lot concerné, changez le statut sur « Pot », puis enregistrez. La date de mise en pot est automatiquement renseignée avec la date du jour (modifiable).

Un lot en pot est généralement en cours de consommation. Le poids peut être mis à jour pour refléter ce qui reste.

✅ Terminé

Quand le lot est vide, passez-le en Terminé. La date de fin est enregistrée automatiquement. Le lot reste visible dans l'historique (bouton Voir terminés) mais n'apparaît plus dans les statistiques actives.

↩️ Réactiver un lot

Un lot terminé peut être réactivé en Cave via le bouton Réactiver. Utile si une erreur a été faite ou si vous retrouvez une boîte oubliée.

Le cycle est cave → pot → terminé mais vous pouvez aussi créer un lot directement en « Pot » ou « Terminé » si vous souhaitez retrouver un historique passé.

Utilisez le bouton 🤖 Auto-compléter en haut du formulaire pour remplir automatiquement les champs via l'IA (nécessite une clé API).

4. Gérer les lots

Depuis la vue détail d'un tabac (cliquez sur une carte dans l'inventaire), vous voyez tous ses lots actifs. En bas à droite : bouton + Lot pour ajouter un nouvel achat.

Ajouter un lot (nouvel achat)

  1. Ouvrez le tabac concerné depuis l'inventaire
  2. Appuyez sur + Lot
  3. Renseignez : statut (Cave), poids, dates, prix, vendeur
  4. Appuyez sur Ajouter
Vous pouvez avoir plusieurs lots du même tabac simultanément à des stades différents — par exemple une boîte en cave achetée pour vieillir et un pot en cours.

Modifier un lot

Chaque lot affiche un bouton ✏️. Modifiez le statut, le poids restant, les dates, puis Enregistrer.

Supprimer un lot

Bouton ✕ sur le lot, confirmation requise. Irréversible.

Afficher les lots terminés

Par défaut, les lots terminés sont masqués. Bouton 👁 Voir terminés (N) pour les afficher dans l'historique.

5. Parcourir l'inventaire

Filtres de statut

Barre de filtre sous la barre de recherche :

  • Tous — tous les tabacs ayant au moins un lot actif (+ tabacs sans lots)
  • Cave — tabacs avec au moins une boîte en cave
  • Pots — tabacs avec au moins un lot en pot
  • Terminés — tabacs avec au moins un lot terminé

Recherche

La barre de recherche filtre sur le nom, la marque, la composition, la catégorie, la coupe et les notes de dégustation. Quand une recherche est active, le filtre de statut est ignoré (affiche tout).

Tri

Menu déroulant : Nom · Marque · Note · Âge (plus ancien en premier) · Quantité (nombre de lots).

Filtres par catégorie / marque

Sur le tableau de bord, cliquez sur une catégorie ou une marque pour filtrer directement l'inventaire.

Statuts d'une pipe

  • Active — pipe en service
  • Finie — pipe retirée (cassée, vendue…). Masquée par défaut, visible via Voir finies.

Bouton ✓ Marquer finie depuis la vue détail. Bouton 🪈 Réactiver pour la remettre en service.

7. Wishlist

Mémo pour les tabacs à acheter. Chaque entrée comporte un nom, une marque et des notes libres.

Bouton 📦 sur un élément de la wishlist : crée une fiche tabac pré-remplie avec le nom et la marque, et retire l'élément de la wishlist.

8. Assistant IA

Disponible dans les formulaires tabac et pipe. Remplissez au moins le nom ou la marque, puis cliquez sur 🔍 Chercher.

L'IA remplit automatiquement : marque, catégorie, composition, coupe (tabac) ou forme (pipe), et description.

Configurer la clé API

  1. Ouvrir les paramètres ⚙️
  2. Section Assistant IA — choisir le fournisseur
  3. Coller la clé API dans le champ
  4. Appuyer sur ✅ Sauvegarder
La clé est stockée uniquement sur votre appareil. Elle n'est jamais transmise à un tiers.

9. Sauvegarde et export

Toutes les options sont dans le menu ⚙️ (icône engrenage, haut à droite).

☁️ Google Drive (recommandé)

Sauvegarde chiffrée dans un dossier privé de votre Google Drive (invisible dans vos fichiers). Authentification OAuth 2.0 directe avec Google.

  • Sauvegarder Drive — écrase la sauvegarde existante
  • Restaurer Drive — affiche un résumé (nombre de tabacs, pipes, date) avant confirmation
La restauration remplace toutes vos données locales. Effectuez un export JSON avant si vous avez un doute.
Sur iOS en mode application (PWA), l'authentification passe par une redirection — vous quitterez brièvement l'application avant d'y revenir automatiquement.

📋 Export JSON

Exporte l'intégralité de vos données (tabacs, pipes, wishlist, clé API) dans un fichier cave-tabac-export.json. Sur iOS, déclenche la feuille de partage. Sur desktop, téléchargement direct.

Ce fichier peut être réimporté via 📂 Importer fichier (.json) ou en le collant dans la zone de texte puis Importer.

📄 Export CSV

Exporte tabacs, pipes et wishlist dans un fichier cave-tabac-export.csv avec BOM UTF-8 (compatible Excel). Chaque ligne correspond à un lot.

📦 Backup ZIP

Archive complète : données JSON + CSV + toutes les images (tabacs et pipes). Idéal pour une sauvegarde locale complète.

📂 Importer fichier (.json)

Sélectionnez un fichier JSON exporté précédemment. Une confirmation affiche le nombre de tabacs et pipes avant d'écraser les données.

10. Mises à jour

Bouton Mises à jour dans les paramètres ⚙️ → Vérifier. Si une nouvelle version est disponible :

  1. Un bandeau affiche la version disponible
  2. Cliquez Mettre à jour
  3. Un avertissement vous invite à sauvegarder d'abord sur Google Drive
  4. Confirmez — l'application se recharge avec la nouvelle version
Effectuez toujours une sauvegarde Google Drive avant une mise à jour. Le cache de l'application est effacé lors du processus.

11. Installation sur l'écran d'accueil

iOS (Safari)

  1. Ouvrir t-cellar.app dans Safari
  2. Appuyer sur l'icône Partager (carré avec flèche vers le haut)
  3. Choisir Sur l'écran d'accueil
  4. Confirmer le nom et appuyer sur Ajouter
L'application fonctionne entièrement hors ligne une fois installée. Les sauvegardes Google Drive nécessitent une connexion internet.

Android (Chrome)

  1. Ouvrir t-cellar.app dans Chrome
  2. Menu ⋮ → Ajouter à l'écran d'accueil
  3. Confirmer
